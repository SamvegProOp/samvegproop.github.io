## ❓ Frequently Asked Questions

### Does this tool delete my files?
No. This tool only interacts with the Windows Shell process. It does not scan, move, or delete your personal documents or application data.

### Why does my screen turn black for a moment?
When the tool restarts the Shell process, the desktop environment must reload. Your icons and taskbar will return after the process succeeds.

### What if the taskbar remains frozen?
If the taskbar does not respond after running the tool, restart your computer. Sometimes the Shell process encounters a lock that a simple restart cannot clear. 

### Is this safe to use?
Yes. The tool uses standard Windows commands that are publicly available. It performs the same actions as the Task Manager restart function but automates the process to save you time.
